```python

# Initialize
```

## 📚 Framework Highlights

### The Cognitive-Scientific Architecture (4 Layers)

Every prompt is built using this proven 4-layer design:

```
Layer 1: DATA CORPUS
  ↓ Authoritative, peer-reviewed sources only
  ↓ IPCC, NOAA, NASA, academic journals

Layer 2: ANALYTICAL TOOLKITS  
  ↓ Formal methodologies & proven techniques
  ↓ Statistical models, ML algorithms, domain frameworks

Layer 3: SYNTHESIS LAYER
  ↓ Multi-source integration & reasoning
  ↓ Cross-domain pattern recognition, uncertainty quantification

Layer 4: OUTPUT GENERATION
  ↓ Communication tailored by persona
  ↓ Personality tone, audience calibration, clarity
```

### The Psychometric Stack (4 Levels)

AI personas are grounded in science:

```
Level 1: ROLE (ISCO-08)
  → Professional anchoring (e.g., Data Scientist = 2165)

Level 2: COGNITIVE FUNCTION (Jungian)
  → Reasoning pattern (e.g., Ti/Ne = Analytical & Creative)

Level 3: PERSONALITY TONE (Big Five)
  → Output style (e.g., Conscientiousness 4.6/5 = Detail-oriented)

Level 4: DOMAIN EXPERTISE
  → Knowledge grounding (IPCC, NOAA, peer-reviewed journals)
```

## 🤝 Contributing

We welcome contributions from prompt engineers, researchers, and enterprises.

### How to Contribute

1. **Read** `CONTRIBUTION-GUIDELINES.md`
2. **Create a feature branch**: `git checkout -b feature/your-prompt`
3. **Add your prompt** with complete metadata
4. **Validate** against schema: `python tools/validate-prompts.py`
5. **Submit PR** with governance sign-off
6. **Pass review** (2+ approvals required)
7. **Merge** to main branch

### Contribution Requirements

- Complete YAML metadata block
- ISCO-08 role code + RIASEC profile
- 18+ behavioral guardrails (must/never/anti-drift)
- 3+ example queries with expected outputs
- Quality score ≥8.0/10
- Governance officer sign-off
