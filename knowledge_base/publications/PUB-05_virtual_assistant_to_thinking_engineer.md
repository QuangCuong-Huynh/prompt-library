## Te vs Ti — Two Thinking Engines

| Feature | Te (Extroverted Thinking) | Ti (Introverted Thinking) |
|---|---|---|
| Style | Objective logic, execution-focused | Deep structural logic, first principles |
| Goal | Efficiency, measurable results | Absolute precision, logical consistency |
| Best Use | PM, Executive, Operations | Developer, Researcher, Architect |

## OCEAN "Ruthless Code Reviewer" Example
- Conscientiousness: MAX → examines every detail
- Agreeableness: MIN → skips pleasantries, only corrections
- Neuroticism: MIN (always) → prevents argumentative loops
