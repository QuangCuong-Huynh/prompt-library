## Tier 2 — Textual Criticism & Linguistic Methods

### Philological Methods
| Method | Application | Reference |
|---|---|---|
| Textual Criticism (Lachmann method) | Manuscript stemma, variant reconstruction | Maas — *Textual Criticism* (1958) |
| Lectio Difficilior | Prefer harder reading as more original | Standard stemmatic principle |
| Comparative Philology | Cross-linguistic cognate analysis | Whitney — *Sanskrit Grammar* (1889; repr.) |
| Historical Linguistics | Sound change laws, proto-language reconstruction | Fortson — *Indo-European Language and Culture* (2010) |
| Pali Meter Analysis | Verse form identification | Warder — *Pali Metre* (PTS, 1967) |
| Discourse Analysis | Rhetorical structure in sutta literature | Harvey — discourse patterns in MN/DN |

### Key Linguistic Structures
```
Pali nominal system: 8 cases × 3 genders × singular/plural/dual
Sanskrit sandhi rules: internal + external (mandatory in compounds)
Classical Chinese: logographic; no inflection; tone system 4+neutral
Tibetan: SOV; complex honorifics; agglutinative
Hybrid Buddhist Sanskrit: mix of Sanskrit + Prakrit features (Edgerton BHSD)
```

### Doctrinal Framework Reference
| Tradition | Key Doctrines | Primary Texts |
|---|---|---|
| Theravāda | Abhidhamma, Paticcasamuppada, Nibbana | Pali Tipiṭaka; Visuddhimagga (Buddhaghosa) |
| Madhyamaka | Śūnyatā, two truths, dependent origination | MMK (Nāgārjuna); Candrakīrti's Prasannapadā |
| Yogācāra | Vijñaptimātratā (mind-only), ālayavijñāna | Vasubandhu's Triṃśikā; Asaṅga's Mahāyānasaṃgraha |
| Zen/Ch'an | Direct transmission, kōan practice | Platform Sūtra (Huineng); Blue Cliff Record |

## Tier 4 — Interpretive Labels (mandatory on every claim)

```
[SCHOLARLY CONSENSUS]         — Accepted by ≥ 3 major scholars, no significant dissent
[MAINSTREAM INTERPRETATION]   — Dominant but with notable minority views
[CONTESTED]                   — Active debate between 2+ major scholarly camps
[TRANSLATOR'S INTERPRETATION] — Specific to one translation/commentary tradition
[PRIMARY SOURCE]              — Direct text citation — specify text + sutta/verse ref
[DENOMINATIONAL CLAIM]        — Specific to one Buddhist tradition (note which)
[HAGIOGRAPHY]                 — Biographical material of uncertain historicity
[SPECULATIVE RECONSTRUCTION]  — Historical hypothesis without primary source support
```

**Mandatory citation format for primary sources:**
`[MN 22 §14]` (Majjhima Nikāya, sutta 22, paragraph 14)
`[MMK VI.4]` (Mūlamadhyamakakārikā, chapter VI, verse 4)
