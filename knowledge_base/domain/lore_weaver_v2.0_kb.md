## Tier 2 — World-Building Methodology

### Sanderson's Laws of Magic (apply to ALL power systems)
```
Sanderson's First Law:
  An author's ability to solve conflict with magic is DIRECTLY PROPORTIONAL
  to how well the reader understands said magic.
  → Hard magic (rules-explicit) resolves plot; soft magic creates wonder.

Sanderson's Second Law:
  Limitations > Powers. What a magic system CANNOT do is more interesting.
  → Always define hard limits before capabilities.

Sanderson's Third Law:
  Expand what you have before adding new elements.
  → Depth beats breadth. One fully realized system > three sketched ones.
```

### POD (Point of Divergence) Protocol
```
Every POD declaration MUST include:
  1. POD_date: [historical date or approximate period]
  2. POD_event: [specific event that diverges]
  3. POD_mechanism: [why/how it diverges]
  4. 1st_order: [immediate consequences within 10 years]
  5. 2nd_order: [consequences within 50 years]
  6. 3rd_order: [structural civilizational changes within 200 years]
  7. butterfly_risks: [elements most likely to collapse under scrutiny]

Format: [POD: {branch} | {date} | {event}]
```

## Tier 4 — Alternate History & Consistency Frameworks

### AH Methodology
| Framework | Key Rule | Application |
|---|---|---|
| POD Declaration | Every counterfactual needs one clear, specific POD | Before any lore writing |
| Butterfly Effect Model | Track 1st/2nd/3rd order consequences | During lore development |
| Turtledove Approach | Near vs. long-term divergence intensity | Calibrating change speed |
| Plausibility Scoring | Rate each element T1–T5 (see Tier 3 scale below) | Canon review |

### Narrative Consistency Protocol (auto-apply)
```
Before writing any lore element, check:
  □ Does it conflict with an existing canon element?
  □ Is the POD clearly established for this branch?
  □ Are 1st/2nd/3rd order consequences tracked?
  □ Does technology level match POD timeline?
  □ Are cultural systems internally consistent?
  □ Is the magic/power system constrained by Sanderson's Laws?

If any box is unchecked → flag [CANON-RISK] and route to Creative Director review
```

### Plausibility Scale (Parallel Worlds internal)
```
[PW-T1] Historical record — documented, verified
[PW-T2] Plausible extension — consistent with historical evidence
[PW-T3] Speculative extrapolation — requires the POD to make sense
[PW-T4] Creative invention — original, internally consistent, not history-backed
[PW-T5] Fantastical — deliberately beyond historical plausibility (magic, myth)
```
